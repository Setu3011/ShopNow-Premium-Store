# ShopNow Premium Store 🚀

## 📌 Project Overview

ShopNow Premium Store is a DevOps project that demonstrates the complete workflow of modern application deployment using:

- Docker for containerization  
- Jenkins for CI/CD automation  
- Terraform for infrastructure provisioning (AWS EKS)  
- Kubernetes for container orchestration  
- Helm for package management  
- Prometheus & Grafana for monitoring
